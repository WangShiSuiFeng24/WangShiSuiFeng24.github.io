安装Visual Studio Community 2022

  1. 下载地址 https://appzip.cn/vs2022/w
  2. 选择社区版下载
  3. 在 工作负荷 中勾选 使用 C++ 的桌面开发
  4. 在 语言包 中勾选 自己习惯的语言，如 中文(简体)
  5. 在 安装位置 中修改安装位置，例如：
    • Visual Studio IDE: D:\Program Files\Microsoft Visual Studio\2022\Community
    • 下载缓存: D:\ProgramData\Microsoft\VisualStudio\Packages
      勾选“安装完成后保留下载缓存”，方便以后更新组件
    • 共享组建、工具和 SDK: D:\Program Files (x86)\Microsoft Visual Studio\Shared
  6. 最后点击 安装 即可。

初次设置

  • 开发设置选择 Visual C++
    (如果点成常规就到 工具 -> 导入和导出设置 -> 重置所有设置 -> 是，保留我的当前设置 里改)
  • 颜色主题选择 深色

Visual C++ 入门

  • 在 Visual Studio 中安装 C 和 C++ 支持
  • 创建 C++ 控制台应用项目
  • 生成并运行 C++ 控制台应用项目
  • 用 C++ 创建控制台计算器

设置vscode Consolas 同款字体

  • 工具 -> 选项 -> 环境 -> 字体和颜色
  • 显示其设置 选择 文本编辑器
  • 字体 选择 Consolas
  • 确定

设置vscode Dark+ 同款主题

法一

  • 扩展 -> 管理扩展 -> 搜索Visual Studio Theme Pack -> 安装扩展极慢（近似没有）

法二

  1. 访问Visual Studio 插件市场
  2. 选择或搜索 Visual Studio Theme Pack
  3. 点击 Download，下载文件名为 VisualStudioThemePack.vsix
  4. 双击 VisualStudioThemePack.vsix 文件，安装插件
  5. 安装完成后，启动 Visual Studio 便可使用。
  6. 启动 Visual Studio 后 工具 -> 主题 -> Dark+
